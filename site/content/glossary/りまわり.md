## 利回りとは

利回り（りまわり）とは、**投資した金額に対して1年間に得られる収益の割合**です。
不動産投資では、家賃収入を物件価格で割った数値として使われます。

## 表面利回りと実質利回り

不動産の利回りには主に2種類あります。

**表面利回り（グロス利回り）**
```
表面利回り = 年間家賃収入 ÷ 物件価格 × 100
```
例：物件価格3,000万円、年間家賃収入180万円 → 表面利回り6%

**実質利回り（ネット利回り）**
```
実質利回り = (年間家賃収入 - 年間諸経費) ÷ (物件価格 + 購入時諸費用) × 100
```
管理費・修繕費・固定資産税などを差し引いた実態に近い利回りです。

## 利回りの目安

| エリア | 表面利回りの目安 |
|---|---|
| 東京都心 | 3〜5% |
| 東京周辺部・大阪 | 5〜7% |
| 地方主要都市 | 7〜10% |
| 地方（空室リスク高） | 10%以上 |

利回りが高い物件は、空室リスクや修繕リスクが高い場合が多いため注意が必要です。

## まとめ

- 表面利回りは概算、実質利回りで実態を把握する
- 利回りだけでなく立地・空室率・築年数も総合判断する
- キャピタルゲイン（売却益）も含めたトータルリターンで評価することが重要
