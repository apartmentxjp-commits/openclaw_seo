仲介手数料

（ちゅうかいてすうりょう）

仲介手数料とは

仲介手数料（ちゅうかいてすうりょう）とは、不動産会社（宅地建物取引業者）が売買・賃貸の仲介を行った際に受け取る報酬です。 法律（宅地建物取引業法）により上限額が定められています。

売買の場合の上限

売買価格に応じた速算式：

  • 200万円以下の部分：売買価格の5%
  • 200万円超〜400万円以下の部分：4%
  • 400万円超の部分：3%

速算式（税別）：売買価格 × 3% + 6万円（400万円超の場合）

例：3,000万円の物件 → 3,000万 × 3% + 6万 = 96万円（税別）

賃貸の場合の上限

賃料の1ヵ月分＋消費税が上限です（借主・貸主合計）。 借主からは、承諾を得た場合のみ1ヵ月分まで請求できます。

仲介手数料の交渉

仲介手数料は上限規制であり、交渉可能です。 特に：

  • 売主と買主の両方を同じ会社が担当する「両手仲介」の場合
  • 高額物件の場合 は値引き交渉の余地があります。

まとめ

  • 売買：物件価格×3%+6万円（400万超・税別）が目安の上限
  • 賃貸：家賃1ヵ月分＋消費税が上限
  • 上限であり交渉可能
  • 物件購入時の諸費用として必ず予算に組み込む
