マンション・管理

管理費

（かんりひ）

管理費とは

管理費（かんりひ）とは、分譲マンションの区分所有者が毎月支払う、共用部分の維持・管理に充てる費用です。 マンション購入後は住宅ローン返済とは別に、毎月の固定費として支払い続けます。

管理費の相場

全国平均は月額1万〜2万円程度です。 タワーマンションや設備の充実した物件では3万円以上になる場合もあります。

管理費の用途

  • エントランス・廊下・エレベーターなど共用部の清掃
  • 共用設備（照明・給排水・エレベーター）のメンテナンス
  • 管理員（コンシェルジュ・管理人）の人件費
  • 火災保険料（共用部分）
  • 管理会社への委託費用

修繕積立金との違い

管理費修繕積立金
目的日常的な維持・管理将来の大規模修繕に備える積立
使い方毎月の管理運営費用12〜15年ごとの大規模修繕等
性質経常的な費用将来への積立

まとめ

  • 管理費は毎月の共用部管理コスト
  • 修繕積立金とセットで購入時のランニングコスト計算に含める
  • 管理費が極端に安いマンションは管理の質に注意
  • 管理組合の財政状況も購入前に確認を
