任意売却

（にんいばいきゃく）

任意売却とは

任意売却（にんいばいきゃく）とは、住宅ローンの返済が困難になった場合に、金融機関の同意を得て不動産を市場で売却する方法です。 競売（裁判所を通じた強制売却）と異なり、一般的な不動産市場で売却できるため、より高い価格での売却が期待できます。

任意売却が必要になるケース

  • 住宅ローンの返済を3〜6ヵ月滞納した
  • 売却価格がローン残高を下回る「オーバーローン」状態
  • 離婚・相続などで物件を手放す必要がある

任意売却と競売の比較

任意売却競売
売却価格市場価格に近い市場価格の50〜70%
プライバシー保たれる公開情報（近隣に知られる）
引越し費用交渉次第で確保可能確保困難
残債一定額を減額交渉できる場合も残債がそのまま残る

任意売却の流れ

  1. 金融機関への相談・同意取得
  2. 不動産会社（任意売却専門）への依頼
  3. 売却活動・価格決定
  4. 売買契約・決済
  5. 残債の処理（債務整理など）

まとめ

任意売却は競売より有利な条件で売却できる可能性が高いです。 返済が困難になる前に、早めに金融機関や専門家に相談することが重要です。
