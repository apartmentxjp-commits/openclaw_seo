住宅ローン

団体信用生命保険

（だんたいしんようせいめいほけん）

団体信用生命保険とは

団体信用生命保険（だんたいしんようせいめいほけん）とは、住宅ローンの返済中に借主が死亡または高度障害状態になった場合に、保険金でローン残債が完済される保険です。 略して「団信（だんしん）」と呼ばれます。

団信の仕組み

  1. 借主が住宅ローンを組む際に加入（多くの場合、加入が融資の条件）
  2. 毎月の保険料は金利に含まれる（別途支払いなしのケースが多い）
  3. 借主が死亡・高度障害になると保険会社がローン残高を一括返済
  4. 遺族はローンなしで住み続けられる

団信の種類と特約

種類保障内容
一般団信死亡・高度障害
三大疾病特約＋がん・急性心筋梗塞・脳卒中
八大疾病特約＋糖尿病・肝硬変など
ワイド団信持病があっても加入できる（金利上乗せ）

注意点

  • 健康状態の告知が必要（持病があると審査落ちの場合も）
  • フラット35は団信が任意加入（加入しない場合は金利優遇なし）
  • 保険料は金利上乗せ方式が多く、0.1〜0.3%程度

まとめ

  • 万が一の際にローン残債がゼロになる重要な保険
  • ほとんどの住宅ローンで加入が必須
  • がん特約など充実した保障を選ぶことも可能
  • 健康告知内容は正確に記載することが重要
